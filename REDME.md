# CSS Box Model Task

## 📌 Project Overview
This project demonstrates the **CSS Box Model** concept using a simple card layout.  
The card contains:
- An image (occupying 85% of the card height).
- A button (occupying 15% of the card height).

The design showcases proper usage of margins, padding, width, height, and gradients.



#Features
Box Model Applied**: `box-sizing: border-box` ensures consistent sizing.
Responsive Layout**: Image and button scale with the container.
Gradient Button**: Stylish button with a linear gradient background.
Semantic HTML**: Clean structure with `<div>`, `<img>`, and `<button>` tags.
Centering with Margin**: Card is positioned using `margin: 100px 250px`.
